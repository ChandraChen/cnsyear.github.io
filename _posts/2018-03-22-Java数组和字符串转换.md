#### Java数组和字符串转换

> 字符串转数组

```Java
String str = "a,b,c,d";
String[] arr = str.split(",");
System.out.println(Arrays.toString(arr));
//输出：[a, b, c, d]
```

> 数组转字符串

```Java
import org.apache.commons.lang.StringUtils;

public class demo {
	 public static void main(String[] args) {
		 String[] arr = {"a","b","c"};
		 
		 String str = StringUtils.join(arr,",");
		 System.out.println(str);
	}
}

//输出：a,b,c
```
