> 需求:文件名在操作系统中不允许出现 / \ ” : | * ? < > 故将其以空替代；

```java
Pattern pattern = Pattern.compile("[\\s\\\\/:\\*\\?\\\"<>\\|]");
Matcher matcher = pattern.matcher(fileName);
fileName= matcher.replaceAll(""); // 将匹配到的非法字符以空替换
```
